## 👨‍💻 About Me

Hey! I'm **Amanuel**, a passionate Software Engineering student based in **Addis Ababa, Ethiopia** 🇪🇹. I love turning ideas into real products — whether it's a full-stack web app, an automation script, or a machine learning model.

I'm currently co-building **[Aurarise](https://github.com/aman-149)** — a SaaS e-learning platform — while diving deep into Full Stack Web Development and ML. I enjoy clean code, thoughtful architecture, and solving real-world problems with technology.
